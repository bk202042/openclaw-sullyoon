## Key extracted claims (excerpt)

- Prompt injection and malicious skill instructions are called out as major risk vectors.
- Security posture recommended: sandbox first, verify restrictions, inspect skills/MCP servers, and use trusted providers for API-keyed integrations.
- Installation/deployment options are framed as trade-offs between convenience and isolation (default local install vs containerized or server-based deployment).
