## Product Thesis
콘텐츠는 조회수보다 재사용 가능한 실행 자산으로 설계한다.

## Product Types
- Mini Guide (PDF)
- Playbook (체크리스트 + 템플릿)
- Toolkit (프롬프트 + 운영시트 + 예시)

## Build Pipeline
1. Source: shared-brain 02_concepts 선택
2. Structure: 문제-해결-실행-검증 템플릿 적용
3. Proof: 실제 사례/수치 삽입
4. Ship: 판매 페이지 + 샘플 공개

## Conversion Support
- 구매자 온보딩 메일
- FAQ/트러블슈팅
- 업데이트 로그
