## Program Goal
참가자가 OpenClaw를 써서 자신의 실제 업무를 자동화 가능한 운영체계로 전환한다.

## Session Design (4 modules)
1. Foundation: OpenClaw 구조와 운영 철학
2. Orchestration: subagent 분해/병렬 실행
3. Assetization: inbox → concepts → projects 변환
4. Operations: 비용/보안/품질 관리

## Hands-on Outputs
- 개인용 shared-brain 초기 세팅
- 본인 업무 1개 자동화 워크플로우
- 재사용 가능한 프롬프트 템플릿 3개

## Evaluation
- 실행 전/후 시간 절감률
- 재사용률(템플릿 재활용 횟수)
- 품질 지표(오류율, 재작업률)
