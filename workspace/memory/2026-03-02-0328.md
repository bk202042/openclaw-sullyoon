### 3. **Telegram allowFrom 설정 오류** (5개 항목)
```
Invalid allowFrom entry: "-1003759829444" 
allowFrom/groupAllowFrom authorization requires numeric Telegram sender IDs only
```
→ Telegram 그룹 ID가 잘못 설정됨. `@username` 대신 **숫자 ID** 필요.

### 4. **OpenAI API Rate Limit**
```
API rate limit reached. Please try again later.
```
→ Codex API 호출 한도 초과. Moonshot(Kimi)으로 폴백됨.

### 5. **Fetch 실패**
```
embedded run agent end: isError=true error=fetch failed
```
→ 외부 API 호출(fetch) 실패. 네트워크 또는 엔드포인트 문제.
