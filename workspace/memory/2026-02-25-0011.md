### 1) 부팅 시 (Boot Refresh)
- 게이트웨이 시작할 때 `qmd update` + `qmd embed` 실행
- **기본값**: 백그라운드로 실행 (채팅 시작 안 막음)
- 동기화 원하면: `memory.qmd.update.waitForBootSync = true` 설정

### 2) 주기적 갱신 (Periodic Update)
- **기본 주기**: **5분마다** (`memory.qmd.update.interval`, default `5m`)
- `qmd update`로 변경된 파일 감지
- `qmd embed`로 벡터 임베딩 생성/갱신

### 설정 가능한 항목
```json5
memory: {
  qmd: {
    update: {
      interval: "5m",        // 갱신 주기
      debounceMs: 15000,     // 파일 변경 후 대기 시간
      onBoot: true,          // 부팅 시 실행 여부
      waitForBootSync: false // 부팅 시 동기/비동기
    }
  }
}
```
